## Default Permission

Allows Takanawa download task commands from the Tauri frontend.

#### This default permission set includes the following:

- `allow-create`
- `allow-start`
- `allow-pause`
- `allow-cancel`
- `allow-snapshot`
- `allow-bitmap`
- `allow-close`
- `allow-download-to-completion`
